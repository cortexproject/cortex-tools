## Installation

## cortextool

```console
# download the binary (adapt os and arch as needed)
$ curl -fSL -o "cortextool" "https://github.com/cortexproject/cortex-tools/releases/download/v0.19.0/cortextool_0.19.0_linux_x86_64"

# make it executable
$ chmod a+x "cortextool"

# have fun :)
$ ./cortextool --help
```
